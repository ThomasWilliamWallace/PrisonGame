Steps to produce visual studio project files:
1. In the `HardTime2 4.26` folder, delete the files generated by CMake, Unreal Engine and Visual Studio.
These are:
	* The folders `.vs`, `Binaries`, `Build`, `Debug`, `DerivedDataCache`, `Intermediate`, `out`, `Saved` 
	* The file `HardTime2.sln`
	* The contents of the `HardTime2 4.26\Source\hardtime_textworld\vs_project` folder
2. Generate a new Visual Studio project from the Unreal Engine code. \
This produces a file `HardTime2.sln`, which can be opened in Visual Studio to load the HardTime2 Unreal Engine C++ project.
To do this, right click on the `HardTime2.uproject` file, and select `Generate Visual Studio Project Files`.
3. Generate the hardtime 'textworld' Visual Studio project from the CMake file.
To do this:
	* Create the `HardTime2 4.26\Source\hardtime_textworld\vs_project` folder if it doesn't already exist.
	* Open a terminal in the `HardTime2 4.26\Source\hardtime_textworld\vs_project` folder.
	* Run `cmake ..`
This produces a file `hardtime_textworld.sln`, which can be opened in Visual Studio to load the hardtime_textworld terminal C++ project.
